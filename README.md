# Cybr - Obsidian Theme

A modern, cyberpunk-inspired theme for Obsidian, based on the [cybrcolors](https://github.com/scherrer-txt/cybrcolors) color palette and design principles from [cybrland](https://github.com/scherrer-txt/cybrland).

## About

This theme brings the vibrant cybrcolors palette to Obsidian, featuring:
- Custom color scheme with cyberpunk aesthetics
- Geist Mono font for better readability
- Dark theme optimized for long coding/writing sessions
- Inspired by the Blackbird theme
